BMI Calculator
Project Description

This project is a GUI-based BMI Calculator developed using Python and Tkinter.

Features
Takes user weight and height input
Calculates BMI
Displays BMI category
Simple and user-friendly interface
Technologies Used
Python
Tkinter
Outcome

The application successfully calculates BMI and displays the health category based on BMI value.
